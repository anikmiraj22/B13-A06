# 🚀 DigiTools - SaaS Product UI

## 📌 Project Overview
DigiTools is a web app that shows off different software products.
It is built using React.
The app lets users browse through digital tools and check out their features. Users can also add products to their cart.The experience of using the app is smooth and interactive.The user interface of DigiTools is designed using Figma.The design is responsive which means it works well on all devices,such, as computers, tablets and smartphones.The DigiTools web app works great on any device you use.

---

## 🛠️ Technologies Used
- ⚛️ React.js
- 🎨 Tailwind CSS
- 🌼 DaisyUI
- 🧠 JavaScript (ES6+)
- 🔔 React-Toastify
- 📦 JSON (for product data)

---

## ✨ Features

### 🛒 Smart Cart System
- Add products to cart instantly
- Remove individual items
- Auto update cart count in Navbar
- Checkout clears all items

### 🔄 Toggle System
- Switch between **Products** and **Cart** view
- Default view shows product list
- Cart shows empty message if no items added

### 🔔 Toast Notifications
- Shows alerts when:
  - Product added
  - Product removed
  - Checkout completed
